Converters

You can find below a few XSLT files that can be used to convert some ERI messages (in XML format) from an edition to another edition. This can be used to detect differences between different editions of the same ERI message. Please note that this is a technical convertion, which may lead to invalid messages in the target edition for functional reasons (missing mandatory fields for example).

Convert ERINOT 1.2f into ERINOT 1.3: ERINOT_1.2f-1.3_v0.2.xslt

Convert ERINOT 1.3 back into ERINOT 1.2f: ERINOT_1.3-1.2f_v0.3.xslt

Github

The ERI messages (XSD format) are available on a public Github. Contributions are protected, but you can freely download the latest stable edition. Previous editions are also available, as well as the edition under development.
