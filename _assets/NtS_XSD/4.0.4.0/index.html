Notices to skipper - XSD Message Namespace

The namespace http://ris.cesni.eu/_assets/NtS_XSD/4.0.4.0/ is referred to as "the XSD message Namespace Edition 4.0".

The namespace contains the release of the message schema. NtS_XSD_V.4.0.4.0.xsd

An html visual version of the message schema is also available under the link Visual XSD edition 4.0.4.0

Release note

In the NtS XSD 4.0.4.0 an error in the regex of the ISRS Location Code (NtS XSD validation) was detected and fixed. The limitation_code_enum maxLength was corrected to be in line with the allowed enumerations.
Further explanation is given in TWG CESNI/TI/NtS/CR200 (see document: cesnitints20_CR200en_Correction_of_NtS_XSD_4.0.4.0_validation_Hotfix_v1.docx).
The NtS XSD 4.0.4.0 has bee updated in this package. Please replace the XSD in your implementation in case of validation errors / use this XSD for new implementations.
