ERIRSP Namespace 2025 (1.4)

Scope and objectives

ES-RIS 2025/1 contains the ERIRSP message in a PDF format. For practical reasons and better readability, the ERIRSP message can also be downloaded here as an XSD file.

This file is provided for information purposes and do not replace the official message which is published in ES-RIS 2025/1.
