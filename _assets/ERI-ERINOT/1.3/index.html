ERINOT Namespace 2021/1 (1.3)

Scope and objectives

ES-RIS 2021/1 contains the ERINOT message in a PDF format. For practical reasons and better readability, the ERINOT message can also be downloaded here as an XSD file.

This file is provided for information purposes and do not replace the official message which is published in ES-RIS 2021/1.
